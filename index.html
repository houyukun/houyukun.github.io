pod install 无法创建 xcworkspace

———>终端显示：Abort trap：6 ———>有Pods文件，但是没有生成xcworkspace 第一: 保证你的‘Podfile’文件格式为 platform :ios, '8.0' use_frameworks! target 'MyApp' do pod 'AFNetworking', '~> 2.6' pod 'ORStackView', '~> 3.0' end 第二: 首先记得替换http： 淘宝升级http协议到https 用这句替换原来的http gem $ gem sources --remove http://ruby.taobao.org/ --add https://ruby.taobao.org/ 成功后输入 $ gem sources -l 可查看到改好的gem源 *** CURRENT SOURCES *** https://ruby.taobao.org/ mac 10.12 cocoapods安装 sudo gem install -n /usr/local/bin cocoapods 第三: 接着：

  • gem install cocoapods 如果不行的话：
  • 升级ruby环境 如果不行的话：
  • sudo gem update --system
  • sudo gem install cocoapods

  • pod setup 如果不行的话：

  • sudo rm -fr ~/.cocoapods/repos/master

  • sudo rm -fr ~/Library/Caches/CocoaPods/
  • pod setup 最后如果不行的话:
  • sudo gem uninstall cocoapods
  • sudo gem install cocoapods --pre 我是这样解决的,安装了两个pod冲突了 注意检查这个Configurations 选中Project->选择Info tab->看看Configurations是不是被设置为 None了，如果是None的话请改为Pods，如图：#
